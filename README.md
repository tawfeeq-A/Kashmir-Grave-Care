# Grave Care Kashmir - React/Next.js Application

A modern, responsive React application for Grave Care Kashmir, built with shadcn/ui components, TypeScript, and Tailwind CSS.

## Project Structure

- app/ - Next.js app directory
- components/ui/ - shadcn/ui components
- pages/ - Next.js pages
- public/ - Static assets
- styles/ - Global styles

## Installation

```bash
cd grave-care-kashmir
npm install
```

## Development

```bash
npm run dev
# Open browser at http://localhost:3000
```

## Build for Production

```bash
npm run build
```

## Features

- Built with shadcn/ui components
- TypeScript for type safety
- Tailwind CSS for styling  
- Framer Motion for animations
- Responsive design with mobile optimization
- Scroll-triggered animations
- Accessibility focused

## Components

### ContainerScroll
A scroll-based animation component using Framer Motion. Includes Header and Card sub-components for building engaging hero sections.

### HeroScrollDemo
Full-screen hero scroll demo demonstrating the ContainerScroll component with image overlay effects.
